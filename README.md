# 飞扬俱乐部官网 (FY-club)

飞扬俱乐部官网是一个展示四川大学飞扬俱乐部形象、服务与项目的现代化网站，以极简设计风格为核心，融合动态交互与视觉美学，打造独特的品牌体验。

## ✨ 设计特色

- **动态视觉效果**：包含Canvas背景动画、悬停变换元素和滚动触发效果，增强页面层次感
- **精致交互体验**：实现翻转卡片展示项目详情、平滑滚动导航和渐变过渡效果
- **双字体系统**：融合「紫霞魂肤瑶手写体」(书法风格)与「紫魂极客黑」(科技风格)，构建独特视觉语言
- **响应式布局**：从移动设备到桌面端的全场景适配，确保一致的用户体验
- **微交互动效**：按钮状态变化、卡片悬浮效果和导航栏滚动变化等细节处理

## 🎯 核心功能模块

1. **首页展示**
   - 动态标题与品牌理念呈现（"创新实践，服务川大"）
   - 响应式导航栏（滚动时变化样式）
   - Canvas动态背景与视觉层次分明的内容区块
   - 滚动触发的元素显示动画

2. **项目展示**
   - 翻转卡片展示项目详情与成果数据
   - 横向滚动的作品展示区
   - 项目分类与案例说明

3. **服务介绍**
   - 品牌标识设计（Brand Identity）
   - 移动应用设计（Mobile Design）
   - UI/UX设计服务
   - 网站设计与开发

4. **客户评价**
   - 交互式客户反馈展示
   - 可翻转查看的项目详情卡片
   - 包含项目时间线与成果数据

5. **联系与社交**
   - 多渠道联系方式与社交平台入口
   - 二维码模态框查看功能
   - 专用B站账号页面与关注入口

## 💻 技术实现

- **前端框架**：Tailwind CSS 构建响应式布局
- **交互逻辑**：原生JavaScript实现动态效果
- **字体处理**：自定义@font-face配置特殊字体（WOFF格式）
- **动画系统**：CSS过渡与变换实现流畅动画效果
- **图标资源**：Font Awesome提供界面图标支持
- **滚动处理**：平滑滚动导航与滚动触发动画

## 📱 B站专区特色

- 俱乐部官方账号信息展示
- 多种互动方式说明（评论、粉丝群、充电支持等）
- 二维码扫码关注功能
- 一键访问B站空间链接
