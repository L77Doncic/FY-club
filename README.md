# 飞扬俱乐部官网 (FY-club)

飞扬俱乐部官网是一个展示四川大学飞扬俱乐部形象、服务与项目的现代化网站。以极简设计风格为核心，融合动态交互与视觉美学，打造独特的品牌体验。

# 💻社团简介

四川大学飞扬俱乐部，创建于2003年9月，至今已成立22年

一个集技术交流、电脑维修、软件研发、网站小程序等建设维护于一体的专业公益IT社团

飞扬秉承“创新实践，服务川大”理念，旨在通过专业技术应用及研发服务大众

向川大师生提供免费的电脑维修维护及其他科技服务

- **🌳发展历史**

2003年 飞扬俱乐部前身飞扬工作室成立

2005年 举办四川师范大学大型维修；四川大学创造协会、公共关系协会正式加入飞扬俱乐部

2006年 协助川大机房检修；完善社团架构；电脑维修量突破1000台

2007年 举办西南财经大学电脑义诊

2008年 举办川大电脑节系列活动；建立飞扬论坛；电脑维修量突破3000台

2009年 建立飞扬俱乐部官网；上线网页版电脑报修系统；电脑维修量突破5000台；

2010年 升级校级社团；创刊《I飞扬》杂志；网页版电脑报修系统升级V2.0

2011年 上线宽带算号系统在线版；实现望江常态大型维修；电脑维修量突破10000台

2012年 入驻成都云公益平台

2013年 成立研发实验室；举办飞扬十周年纪念活动

2016年 上线会员验证系统；上线绩点计算器

2017年 上线 “we川大”小程序；上线绩点计算器3.0；更新报修系统

2018年 上线微信报修小程序；与西南民族大学交流合办大型维修

2019年 与西华大学点击网络交流

## ✨ 设计特色

- **动态视觉效果**：包含Canvas背景动画、悬停变换元素和滚动触发效果，增强页面层次感
- **精致交互体验**：实现翻转卡片展示项目详情、平滑滚动导航和渐变过渡效果
- **微交互动效**：按钮状态变化、卡片悬浮效果和导航栏滚动变化等细节处理
